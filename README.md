# ISMB 2026 Talks Explorer

Browse the ISMB 2026 talks (and co-located ICBO) by track, search them, and star the ones you don't want to miss. It's one self-contained HTML file that runs offline in any browser.

**Get it:** open [ISMB2026-Talks-Explorer.html](ISMB2026-Talks-Explorer.html), click **Download raw file**, then open the downloaded file in your browser.

Talks only — posters aren't listed (ISCB doesn't publish them openly). Data is pulled straight from the official schedules, snapshotted 2026-07-14. Stars are saved in your browser.

Repurposed from Woominsong's ICML 2026 skill — thanks for sharing it.

Not affiliated with ISCB/ISMB.
